# Parfum E-Commerce Frontend

React/Vite client for the Parfum E-Commerce prototype. It provides the storefront, authentication forms, persistent cart, demo checkout, and administrator catalogue pages.

## Development

Run the API from `../server`, then:

```bash
npm install
npm run dev
```

The development server proxies `/api` to `http://localhost:5000`.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

See the [root README](../README.md) for full setup instructions and known security limitations.
